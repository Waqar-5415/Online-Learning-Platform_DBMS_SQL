# Online Learning Platform (DBMS Project)

## Database
MySQL

## Project Description
This project is a database-driven Online Learning Platform (LMS with Analytics) designed using core DBMS concepts including ER modeling, normalization (up to 3NF), constraints, and advanced SQL queries.

## Features
- User management (Students, Instructors, Admin)
- Course management
- Enrollments
- Assignments and submissions
- Performance analytics

## Files Included
- ERD.png – Entity Relationship Diagram
- database_script.sql – Complete MySQL database implementation
- Project_Report.pdf – Final project report

## How to Run
1. Open MySQL Workbench
2. Run `database_script.sql`
3. Database will be created automatically
